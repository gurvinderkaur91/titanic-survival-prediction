Titanic Survival Prediction System

A Machine Learning project developed to predict whether a passenger would survive the Titanic disaster based on passenger class, gender, age group, fare category, embarkation point and family information. The project includes data cleaning, feature engineering, model training, evaluation and deployment using Streamlit.

⚙️ Project Workflow

1. Data Collection

Collected Titanic passenger dataset.

2. Data Cleaning

Removed unnecessary columns, handled missing values and cleaned records.

3. Feature Engineering

Created Age Groups and Family Members features.

4. Exploratory Data Analysis

Analyzed survival trends based on gender, class and embarkation port.

5. Machine Learning

Trained classification model for survival prediction.

6. Deployment

Deployed the final model using Streamlit.
